Описание проекта

Цель:
Разработка консольного приложения для получения информации о погоде по указанному городу или текущему местоположению. Приложение также будет хранить историю запросов с возможностью её вывода.

Реализация:
1. Получение информации о погоде по названию города:

- Используется OpenWeatherMap API, предоставляющее данные о текущих погодных условиях.
- Для взаимодействия с API применяется библиотека requests.
- Данные запрашиваются в формате JSON, с параметрами:
- q (название города),
- appid (API-ключ),
- units (метрическая система),
- lang (язык, например, ru для русского).

2. Определение текущего местоположения и получение погоды:

- Для геолокации используется библиотека geopy (сервис Nominatim). С помощью неё определяется текущий город или координаты по IP-адресу пользователя.
- Данные о погоде для текущего местоположения также запрашиваются через OpenWeatherMap API, но с использованием широты (lat) и долготы (lon) вместо названия города.

3. Хранение истории запросов:

- История запросов сохраняется в оперативной памяти программы в виде списка Python. Каждая запись включает:
- Дату и время запроса.
- Название города.
- Погодные условия (температура, ощущаемая температура, скорость ветра и пр.).
- Для сохранения истории между запусками может быть использован файл формата JSON.

4. Интерфейс программы:

- Реализуется консольный интерфейс с выбором действий:
- Запрос погоды по названию города.
- Запрос погоды по текущему местоположению.
- Вывод последних N запросов из истории.
- Завершение работы программы.
- Пользователь взаимодействует с приложением через меню, выбирая опции по номеру.

5. Обработка ошибок и исключений:

- Используется конструкция try...except для предотвращения сбоев:
- Обработка ошибок сети и недоступности API.
- Обработка ошибок некорректного ввода (например, несуществующее название города).
- Обработка ответа API с ошибками (например, неверные параметры запроса).
- При ошибке пользователю выводится понятное сообщение, а программа продолжает работу.
